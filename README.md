# 放松阅读 

中文 | [English](./README-en.md)

简单的桌面txt阅读器，基于tkinter开发。适合桌面阅读小说，可完美融入多数背景。

## 软件界面

![avatar](./imgs/demo.jpg)

## 使用方式

1. 下载 `read.py` 和 `config.json` 至txt文件所在目录。

2. 打开cmd执行 `python read.py`

或者

1. 下载 `read.exe` 和 `config.json` 至txt文件所在目录。

2. 双击执行 `read.exe`

## 操作说明

| 快捷键 | 说明 |
| ------ | ------ |
| ← | 上一页 |
| → | 下一页 |
| ↑ | 显示窗口 |
| ↓ | 隐藏窗口 |
| ctrl + ↓ | 退出软件 |
| 滚轮 ↑ | 上一页 |
| 滚轮 ↓ | 下一页 |
| 鼠标左键 | 按住拖动窗口 |
| 鼠标右键 | 打开菜单 |
