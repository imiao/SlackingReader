# 放松阅读 SlackingReader

简单的桌面txt阅读器，基于tkinter开发。

Simple txt reader for dektop, developed base on tkinter.

## 使用方式

1. 下载read.py和config.json至txt文件所在目录。

2. 打开cmd执行 `python read.py`

3. 操作说明：

| 快捷键 | 说明 |
| ------ | ------ |
| ← | 上一页 |
| → | 下一页 |
| ↑ | 显示窗口 |
| ↓ | 隐藏窗口 |
| ctrl + ↓ | 退出软件 |
| 滚轮 ↑ | 上一页 |
| 滚轮 ↓ | 下一页 |
| 鼠标左键 | 按住拖动窗口 |
| 鼠标右键 | 打开菜单 |
